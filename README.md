# Shelfly

AI의 힘으로 1-2주만에 웹을 ‘달칵’ 만들어보자!


#### 개인 맞춤 독서 기록 및 추천 사이트

- **주제**: 독서 기록을 아카이빙하고 AI가 취향에 맞춘 책을 추천해 주는 사이트
- **기능**: 읽은 책을 기록하고, 각 책에 대한 감상평이나 메모를 남길 수 있는 공간.

#### 사용한 AI 툴

- 프론트: v0
- 백: GPT 4o, claude
- IDE: Cursor AI
